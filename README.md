# Credit Card Risk & Behavior Analysis

This project explores customer-level credit card behavior and default risk using
a combination of Kaggle datasets and World Bank API data.

## Structure
- pages/: Streamlit app pages (EDA, PCA, clustering, models)
- src/: Data loading and API scripts
- data/: Local datasets (not all included)

## Data Sources
- Kaggle: Credit Card User Behavior Dataset
- World Bank API: Domestic credit to private sector (% GDP)

## How to run
pip install -r requirements.txt  
streamlit run Home.py
